## Universal Skill Assessment Levels

### Level 1: Awareness (Beginner)
**Hours**: 0-20 | **Depth**: Surface-level | **Confidence**: Low

#### Knowledge Indicators
- [ ] Can define key concepts
- [ ] Knows major terminology
- [ ] Understands basic theory
- [ ] Recognizes patterns

#### Practice Indicators
- [ ] Knows basic syntax
- [ ] Can follow tutorials step-by-step
- [ ] Completes guided exercises
- [ ] Understands examples

#### Assessment Methods
- Multiple choice questions (70%+ to pass)
- Vocabulary/definition matching
- Basic concept identification
- Video comprehension checks

#### Example Tasks
- "What is REST?"
- "Explain the MVC pattern"
- "Identify the three main types of databases"

---

### Level 2: Beginner (Entry-Level Ready)
**Hours**: 20-80 | **Depth**: Foundational | **Confidence**: Moderate

#### Knowledge Indicators
- [ ] Understand core concepts

- [ ] Grasp theory behind tools
- [ ] Recognize best practices

#### Practice Indicators
- [ ] Write simple code independently
- [ ] Solve guided problems
- [ ] Apply basic patterns
- [ ] Follow established conventions

#### Assessment Methods
- Coding exercises (simple)
- Practical problem-solving
- Mini-project creation
- Code review feedback

#### Example Tasks
- "Build a simple CRUD API"
- "Write a function to sort an array"
- "Create a database schema for a blog"

## Assessment Methods by Level

### Level 1: Awareness
- **Duration**: 30 min - 1 hour
- **Format**: Multiple choice, true/false
- **Pass Rate**: 70%+
- **Tools**: Quiz, video comprehension

### Level 2: Beginner
- **Duration**: 1-2 hours
- **Format**: Coding exercises, guided projects
- **Pass Rate**: Functional code
- **Tools**: Coding platform, automated tests

### Level 3: Competent
- **Duration**: 4-8 hours
- **Format**: Real-world project
- **Pass Rate**: Production-ready code
- **Tools**: Code review, system design

### Level 4: Proficient
- **Duration**: 1-2 days
- **Format**: System design, architecture
- **Pass Rate**: Expert feedback
- **Tools**: Interview panels, code review

### Level 5: Expert
- **Duration**: Ongoing
- **Format**: Publications, contributions
- **Pass Rate**: Peer recognition
- **Tools**: Publication, community
