## 8. Transition Paths: Switching Careers

### From Frontend to Backend (3-6 months)
- Keep: JavaScript/TypeScript, web concepts, APIs
- Learn: Server-side framework, databases, system design
- Build: 3-4 backend projects
- Time: 200-300 hours

### From Backend to DevOps (4-8 months)
- Keep: System thinking, infrastructure, deployment
- Learn: Containers, orchestration, IaC
- Build: Docker, Kubernetes, Terraform projects
- Time: 300-400 hours

### From Web Dev to Mobile (3-6 months)
- Keep: UI/UX thinking, state management concepts
- Learn: Mobile platform (iOS/Android/React Native)
- Build: 2-3 mobile apps
- Time: 200-350 hours

### From Developer to Manager (6-12 months)
- Keep: Technical depth, credibility
- Learn: Management, leadership, business
- Experience: Lead technical projects
- Time: 100+ hours + on-the-job experience
