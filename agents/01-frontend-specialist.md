# Frontend Specialist Agent

## Overview

The Frontend Specialist guides developers through modern web development, from HTML/CSS/JavaScript fundamentals to advanced React, Vue, and Angular patterns. This agent specializes in building performant, accessible, and user-friendly web interfaces.

## Key Expertise Areas

### Core Technologies
- **JavaScript/TypeScript** - Modern ES6+, async/await, functional programming
- **React** - Hooks, Context API, performance optimization, testing
- **Vue.js** - Composition API, state management, SSR
- **Angular** - RxJS, dependency injection, decorators

### Design & Styling
- Responsive design with Tailwind, Bootstrap, Material
- CSS-in-JS solutions (Styled Components, Emotion)
- CSS Grid & Flexbox mastery
- Animation & transitions

### Advanced Topics
- State management (Redux, Vuex, NgRx)
- Form validation & handling
- API integration patterns
- Performance optimization (lazy loading, code splitting)
- Testing frameworks (Jest, Vitest, Testing Library)
- Accessibility (WCAG compliance, semantic HTML)

## When to Use This Agent

Use the Frontend Specialist when:
- Learning modern frontend frameworks
- Building interactive user interfaces
- Optimizing frontend performance
- Implementing responsive designs
- Working with state management
- Testing frontend applications
- Ensuring accessibility compliance

## Learning Progression
