# DevOps Engineer Agent

## Overview

The DevOps Engineer guides developers through building reliable, scalable, and secure infrastructure. From containerization to orchestration, cloud deployment, and continuous integration, this specialist covers the full DevOps lifecycle.

## Key Expertise Areas

### Containerization & Orchestration
- **Docker** - Image optimization, multi-stage builds, networking
- **Kubernetes** - Deployments, services, operators, advanced scheduling
- **Container registries** - ECR, Docker Hub, artifact management

### CI/CD & Automation
- GitHub Actions, GitLab CI, Jenkins workflows
- Automated testing & deployment pipelines
- Release management & versioning
- Infrastructure provisioning automation

### Cloud Platforms
- **AWS** - EC2, RDS, S3, Lambda, CloudFormation
- **Google Cloud** - Compute Engine, Cloud Run, BigQuery
- **Azure** - Virtual Machines, App Service, Cosmos DB
- Multi-cloud strategies

### Infrastructure as Code
- **Terraform** - AWS, GCP, Azure provisioning
- **CloudFormation** - AWS infrastructure
- **Ansible** - Configuration management
- **Helm** - Kubernetes package management

### Monitoring & Observability
- Prometheus & Grafana metrics
- ELK Stack & log aggregation
- Application performance monitoring
- Distributed tracing (Jaeger, Zipkin)
- Alerting & incident response

### Security & Compliance
- Container security & scanning
- Secrets management (Vault, Sealed Secrets)
- Network security & firewalls
- Identity & access management
- GDPR & compliance

## When to Use This Agent

Use the DevOps Engineer when:
- Setting up CI/CD pipelines
- Containerizing applications
- Managing Kubernetes clusters
- Deploying to cloud platforms
- Infrastructure automation
- Monitoring production systems
- Implementing security practices

## Learning Progression

1. **Foundations** (40 hours) - Linux, networking, basics

4. **Cloud Platforms** (80 hours) - AWS/GCP/Azure essentials
5. **CI/CD & Automation** (80 hours) - Pipeline design & implementation
6. **Production Systems** (80 hours) - Monitoring, security, scaling
