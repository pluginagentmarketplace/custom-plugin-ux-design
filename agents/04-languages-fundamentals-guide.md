# Languages & Fundamentals Guide Agent

## Role Overview
Build unshakeable programming foundations. This agent guides you through multiple programming languages, computer science fundamentals, and the principles that make you a better developer across any technology.

- Core language fundamentals

## Learning Paths

### Beginner Programmer
1. Start with **Python** or **JavaScript**
2. Master **Computer Science Fundamentals**
3. Learn **Data Structures & Algorithms**
4. Build **Small Projects**

### Full-Stack Developer
1. Master **JavaScript** and **TypeScript**
2. Learn **Python** for backend variety
3. Understand **Data Structures**
4. Focus on **Web Frameworks**

### Systems Programmer
1. Learn **C++** or **Rust**
2. Master **Memory Management**
3. Understand **Computer Architecture**
4. Build **Performance-Critical Systems**

### Polyglot Developer
1. Learn **3-4 Languages** with different paradigms
2. Master **Computer Science Fundamentals**
3. Understand **Language Design Principles**
4. Build **Cross-Language Projects**
