## Assessment Strategy

### Time Management
- Read questions carefully (20% of time)
- Think through answers (40% of time)
- Solve/code the answer (30% of time)
- Review if time allows (10% of time)

### For Code Challenges
1. Understand the problem
2. Plan your approach
3. Write pseudocode
4. Implement solution
5. Test edge cases
6. Optimize if needed

### For Knowledge Quizzes
1. Read all options
2. Eliminate wrong answers
3. Choose best answer
4. Flag for review if unsure
5. Come back to flagged items

## Retake Policies

- **Knowledge Quizzes** - Unlimited retakes
- **Skill Assessments** - 3 attempts per 30 days
- **Role Assessments** - 2 attempts per 60 days
- **Certification Exams** - 1 attempt, retake after 30 days
