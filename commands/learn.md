# /learn

Start your personalized learning journey with step-by-step guidance from specialized agents.

## Usage

```
/learn [role] [--restart] [--track-progress]
```

## Examples

### Start learning a new role
```
/learn react-developer
/learn backend-python
/learn devops-kubernetes
```

### Resume learning from where you left off
```
/learn react-developer --resume
```

### Reset progress and restart
```
/learn react-developer --restart
```

### Track progress automatically
```
/learn react-developer --track-progress
```

## How It Works

1. **Assessment** - Quick evaluation of current knowledge level
2. **Personalized Path** - Custom learning sequence based on your level
3. **Agent Guidance** - Dedicated specialist guides you through each topic
4. **Interactive Learning** - Code examples, exercises, and challenges
5. **Progress Tracking** - Automatic tracking of completed topics

## Learning Stages

### Stage 1: Foundations (40-60 hours)
- Core concepts and fundamentals
- Essential tools and environments
- First practical project
- Checkpoint assessment

### Stage 2: Intermediate Skills (60-100 hours)
- Advanced patterns and practices
- Real-world scenarios
- Building complete projects
- Mid-way evaluation

### Stage 3: Advanced Expertise (80-120 hours)
- Complex system design
- Performance optimization
- Best practices and conventions
- Portfolio-quality projects

### Stage 4: Professional Mastery (40-80 hours)
- Industry standards and trends
- Advanced problem-solving
- Mentoring and teaching
- Career advancement strategies

## Recommended Pace

**Full-Time Learning**
- 3-6 months to proficiency
- 8-10 hours per week
- Complete from foundations to advanced

**Part-Time Learning**
- 6-12 months to proficiency
- 4-6 hours per week
- Flexible scheduling

**Self-Directed Learning**
- Variable pace
- Choose your own topics
- Customize the path

## Tips for Success

1. **Be Consistent** - Regular practice beats cramming
2. **Do the Projects** - Build real things, not just read
3. **Ask Questions** - Use agents to clarify concepts
4. **Review Regularly** - Reinforce learning
5. **Build Portfolio** - Show what you've learned
6. **Network** - Connect with other learners
7. **Teach Others** - Solidify your knowledge
