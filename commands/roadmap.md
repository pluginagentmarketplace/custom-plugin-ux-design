# /roadmap

Explore developer career roadmaps and learning paths from the comprehensive Developer Roadmap collection covering 65+ roles.

## Usage

```
/roadmap [category] [role]
```

## Examples

### Browse all categories
```
/roadmap
```

### Explore a specific category
```
/roadmap frontend
/roadmap backend
/roadmap devops
/roadmap data-science
/roadmap mobile
/roadmap databases
/roadmap product-management
```

### Get a detailed roadmap for a specific role
```
/roadmap frontend react-developer
/roadmap backend nodejs-developer
/roadmap devops kubernetes-engineer
```

## Available Categories

### Frontend Development
- **React Developer** - Master modern React patterns, hooks, and performance
- **Vue Developer** - Vue.js fundamentals and advanced patterns
- **Angular Developer** - Enterprise Angular development
- **Web Developer** - Full-stack web development
- **CSS Expert** - Advanced CSS and styling techniques

### Backend Development
- **Node.js Developer** - JavaScript backend with Express, Fastify, NestJS
- **Python Developer** - Django, FastAPI, Flask ecosystems
- **Java Developer** - Spring Boot and enterprise patterns
- **Go Developer** - Systems programming and CLI tools
- **Rust Developer** - Systems and performance programming

### DevOps & Infrastructure
- **DevOps Engineer** - Docker, Kubernetes, CI/CD pipelines
- **Cloud Architect** - AWS, GCP, Azure cloud design
- **SRE (Site Reliability Engineer)** - System reliability and automation
- **Kubernetes Specialist** - Kubernetes deep expertise

### Data & AI
- **Data Engineer** - Data pipelines and ETL
- **Data Scientist** - Machine learning and analytics
- **AI Engineer** - Large language models and AI applications
- **ML Engineer** - Machine learning systems

### Mobile Development
- **iOS Developer** - Swift and native iOS development
- **Android Developer** - Kotlin and native Android development
- **React Native Developer** - Cross-platform mobile with React
- **Flutter Developer** - Dart and Flutter framework

### Databases & Data
- **SQL Developer** - PostgreSQL, MySQL, relational databases
- **NoSQL Developer** - MongoDB, DynamoDB, document databases
- **Database Architect** - Database design and optimization
- **Redis Specialist** - Caching and in-memory databases

### Product & Growth
- **Product Manager** - Product strategy and management
- **UX/UI Designer** - User experience and interface design
- **Growth Hacker** - Growth strategies and acquisition
- **Data Analyst** - Analytics and insights

## Features

- **Detailed Learning Paths** - Step-by-step progression from basics to advanced
- **Time Estimates** - Expected hours for each skill
- **Related Roles** - See complementary career paths
- **Project Ideas** - Hands-on practice projects
- **Resource Links** - Curated learning materials
- **Skill Prerequisites** - Required knowledge before starting
